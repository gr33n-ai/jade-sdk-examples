# Jade SDK Examples

Example applications demonstrating how to use the [Jade SDK](https://docs.gr33n.ai) to build AI-powered apps with image, video, and audio generation.

## Examples

| Example | Description | Status |
|---------|-------------|--------|
| [React Native](./react-native) | Expo app with chat UI, streaming, and media generation | Ready |
| [Next.js](./nextjs) | Web app with server-side streaming | Coming Soon |

## Quick Start

### React Native

```bash
cd react-native
npm install
npm start
```

See [react-native/README.md](./react-native/README.md) for detailed setup instructions.
